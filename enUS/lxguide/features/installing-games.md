# Game Setup
LauncherX provides different methods of setting-up game clients.

[[toc]]

## By Downloading

With direct downloading provided by LauncherX, you can install games by version with optinal affiliated component (such as Forge, Fabric). Steps:
* Go `Download` on Navbar
* Find or search a game version, then click `Download`
* Select one or more game (or moded game) on pop-up window
    * Select the first item on the list then `Install` if you only install vanilla game
    * Or select any optional affiliated components, then choose any version you want for them. Go `Install`.

    ::: tip Info

    With the powerful task queue management in LauncherX, you can select multiple games and install parallelly.

    Please note that optional affilated components would **not work** if the corresponding game version is not installed.

    You can modify `game name` on the installation pop-up.

    :::

* Wait until it finish. If installation fails, please check your Internet connection. Check: [No Eng! 管理账户 | 日冕知识库](/zhCN/lxguide/settings/special/manage-accounts.html)

## Modpack Installation
LauncherX supports **Modrinth** and **Curseforge** format modpack. Simply drag and drop modpack into LauncherX window (or go`Game List`, find `More - Add game resources - Modpack`) and wait for installation. Learn more: [Not Eng! LauncherX的整合包支持 | 日冕知识库](/zhCN/lxguide/features/modpack-support.html#导入)

## Migrate Game
You can manually add any directory as a game directory in LauncherX. LauncherX can only identify a common isolated game version (not like a version from MultiMC). Or you can copy **an isolated version** (usually in path `.minecraft/versions/`) and paste in `versions` folder which in a game directory that is already added in LauncherX. Copy and paste method might be not work. Check: [No Eng! 游戏列表 | 日冕知识库](/zhCN/lxguide/features/gamelist.html)
